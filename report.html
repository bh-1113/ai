📂 보고서 자동 생성

ChatGPT로 Word, Excel, PowerPoint 형식의 보고서를 만드는 방법을 배워보세요!

알림: GitHub Pages 같은 정적 사이트에서는 버튼으로 파일을 '자동 생성'할 수 없습니다. 자동 생성을 원하면 별도 백엔드(예: Render + FastAPI)가 필요합니다.

📝 Word 보고서

ChatGPT에게 문서를 작성시키고, 결과를 Word(.docx)로 저장하세요.

예시 프롬프트:

“AI가 교육 분야에 미치는 영향”이라는 주제로 보고서를 작성해줘.
- 목차 포함
- 2000자 내외
- 마지막에 참고문헌 섹션 추가
📥 Word 파일 다운로드 (예시)

📊 Excel 보고서

표 데이터를 생성한 뒤 Excel(.xlsx)에 옮겨 활용할 수 있습니다.

예시 프롬프트:

“2025년 매출 목표 달성을 위한 월별 예산표”를 작성해줘.
항목: 광고비, 인건비, 서버비용, 마케팅
형식: 표 형태
단위: 만원
📥 Excel 파일 다운로드 (예시)

📑 PowerPoint 보고서

슬라이드 목차와 요약을 받아 PPT로 정리하세요.

예시 프롬프트:

“AI와 자동화의 미래” 주제로 발표 자료를 만들어줘.
- 10장의 슬라이드
- 각 슬라이드 제목과 요약 내용 포함
- 발표용 원고 추가
📥 PPT 파일 다운로드 (예시)
